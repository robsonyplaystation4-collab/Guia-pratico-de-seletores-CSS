Guia prático de seletores CSS

Exemplos visuais para aula introdutória de seletores

1. O problema inicial

Em CSS, o seletor responde à pergunta: em quais elementos esta regra será aplicada?

2. Seletor universal e seletor por tipo

Este parágrafo recebe estilo do seletor por tipo p. 
* {
  box-sizing: border-box;
}

p {
  color: #334155;
}

3. Classe e ID

Este parágrafo usa a classe .destaque. 
.destaque {
  background: #eef6ff;
  border-left: 4px solid #2563eb;
}
Este parágrafo usa o ID #aviso-unico. 
#aviso-unico {
  color: #b45309;
  font-style: italic;
}

4. Combinadores

  • Início
  • Contato
nav a {
  color: #1d4ed8;
}
  • Item A
  • Item B
    • Subitem
ul.menu-filho > li {
  background: #eef2ff;
}

5. Pseudo-classes

Passe o mouse aqui 
.botao:hover {
  background: #1d4ed8;
}
  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemento 4
.zebra li:nth-child(2n) {
  background: #e0f2fe;
}

6. Pseudo-elementos

Este texto recebe conteúdo antes e depois.
.aviso::before {
  content: ">> ";
}

.aviso::after {
  content: " <<";
}

7. Conflito entre regras e especificidade

Este texto recebe seletor por tipo, por classe e por ID. 
p { color: #334155; }
.destaque { color: #0ea5e9; }
#principal { color: #991b1b; }

O ID vence por ser mais específico.

8. Síntese

  • Seletores escolhem elementos
  • Classe é reutilizável
  • ID deve ser único
  • Especificidade resolve conflitos
